
# Carousel Assignment

This is basic react app for Carousel.




## Run Locally

Clone the project

```bash
  git clone https://github.com/dark-venom26/Carousel-Assignment
```

Go to the project directory

```bash
  cd Carousel-Assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
